Classes
=======

- :py:class:`BaseSheep`:
  A base class for container adapters - classes that allow launching different kinds of containers.

- :py:class:`DockerSheep`:
  Sheep running its jobs in docker containers.

- :py:class:`BareSheep`:
  An adapter that running models on bare metal with ``cxworker-runner``.

- :py:class:`SheepError`:
  Common base class for all exceptions

- :py:class:`SheepConfigurationError`:
  Common base class for all exceptions
