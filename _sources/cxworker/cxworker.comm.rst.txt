Classes
=======

- :py:class:`InputMessage`:
  Message informing the runner about a job being ready to be processed.

- :py:class:`DoneMessage`:
  Message informing :py:class:`cxworker.shepherd.Shepherd` about a finished job.

- :py:class:`ErrorMessage`:
  Message informing :py:class:`cxworker.shepherd.Shepherd` about an encountered error.

- :py:class:`Messenger`:
  Static helper class for sending and receiving messages through zmq sockets.

- :py:class:`JobDoneNotifier`:
  Simple notifier build over green zmq PUB-SUB sockets.
